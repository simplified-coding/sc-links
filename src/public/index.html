Simplified Coding Link Manager

Welcome to the Simplified Coding Link Manager. This website itself doesn't have much around here, but it's the home for all kinds of links that Simplified Coding uses.

Try using it, by typing a slug after https://links.simplifiedcoding.org/.
For example, https://links.simplifiedcoding.org/discord will redirect you to an invitation for our Discord server.

You can check out our Airtable, where you can find all the slugs and their corresponding websites they redirect to.
